    Training Layer¶

    Loads labeled feature vectors, splits them into train/val/test sets, and fits a model. Parallel to the inference layer — neither imports from the other.

    Consumes: FeatureVector parquet files (written by FeaturePipeline) + researcher-supplied labels.csv

    Emits: Saved model directory (via model.save())

    src/ml4em/training/
  base.py       Trainer Protocol
  dataset.py    FeatureDataset — load features + join labels
  trainer.py    StandardTrainer

    Trainer Protocol¶

    The contract every trainer must satisfy.

    class Trainer(Protocol):
    def fit(self, dataset: FeatureDataset) -> None: ...
    def save(self, path: str) -> None: ...

    FeatureDataset¶

    Loads FeatureVector objects from parquet files and joins them with a researcher-supplied labels CSV.

    Consumes: Parquet feature files + labels.csv

    Emits: list[LabeledSample] — FeatureVector + integer label, joined on source_id

    from ml4em.training import FeatureDataset

dataset = FeatureDataset.from_storage(cfg.storage, labels_path="labels.csv")

print(dataset.class_counts())       # {0: 5000, 1: 320}
print(dataset.positive_fraction())  # 0.060

train, val, test = dataset.split(val_fraction=0.1, test_fraction=0.1, seed=42)

    Labels CSV format¶

    source_id,label
686149073900013696,1
686149073900013697,0
    • source_id: same string ID as LightCurve.source_id
    • label: 1 = positive class, 0 = background
    • Sources present in only one of (features, labels) are silently skipped

    Labels are never generated by ml4em. See Preparing Labels for sourcing them from external catalogs.

    StandardTrainer (shell)¶

    Orchestrates training: calls model.fit(train, val) and delegates save to model.save(path).

    Consumes: FeatureDataset + MLModel

    Emits: Saved model directory

    from ml4em.training import StandardTrainer

trainer = StandardTrainer(model, cfg.training)
trainer.fit(dataset)   # NotImplementedError — training loop pending
trainer.save(path)     # delegates to model.save(path) — implemented

    cfg.training parameters¶

    Parameter Description
    learning_rate Step size for gradient descent (neural net trainers)
    batch_size Samples per gradient update
    max_epochs Maximum training epochs
    patience Early stopping: halt after this many epochs with no val improvement
    val_fraction Fraction of data held out for validation
    test_fraction Fraction of data held out for final evaluation
    seed Random seed for reproducibility

    Status: fit raises NotImplementedError — training loop pending model architecture finalization. save is fully implemented.
